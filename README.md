# Magical-Drop-2-Remaster
A recreation of Magical Drop 2 in C++ using SFML. To run the program, open Executable/LastTry.exe. To view the source code, see Source/main.cpp. As this was my first c++ project i only kept one file to keep it simple.

For this project I learned how to use the SFML library to display the game. As I developed the project I learned new ways to code and debug. This was one of the first and most valuable projects i created at university.
